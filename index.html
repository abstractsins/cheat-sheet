Order of Pseudo-Class Selectors

Since later CSS styles override any styles that come before, there is a preferred order to pseudo-class selectors: (Notice the different ways to specify color)


.class a:link {
color: rgb(248, 99, 99);
}

.class a:visited {
color: #A020F0;
}

.class a:focus {
color: hsl(174, 100%, 50%);
}

.class a:hover {
color: deeppink;
}

.class a:active {
color: var(--active-color);
}

Go-To Array Methods

The following is a curated list of useful go-to array methods. Access the reference URL for more methods.

(Reference: developer.mozilla.org)


arrayName.push('String', numb3r, or someVariable);
-- adds the passed argument to the end of the array

arrayName.pop();
-- removes the last item in an array

arrayName.splice(index, span);
-- removes item(s) at specified index, for the length of span

arrayName.length();
-- returns the number of items in an array

arrayName.indexOf('String')
-- Returns the first (least) index of an element within the array equal to an element

arrayName.forEach((element) => {function body});
-- Iterator, calling a function on each element in the array

let arrayName = object.keys();
-- creates new array, populated by an object's keys

let arrayName2 = arrayName1.filter(functionCall);
-- creates a new array by iterating over an array and evaluating each item as TRUE or FALSE
