Self Closing Tags

The following tags, called "void-elements," are self closing. The slash at the end "/>" is only required for XHTML, but it is recommended to always use it.

(Reference: http://xahlee.info/js/html5_non-closing_tag.html)


<area />
<base />
<br />
<col />
<embed />
<hr />
<img />
<input />
<link />
<meta />
<param />
<source />
<track />
<wbr />

>> CSS

Order of Pseudo-Class Selectors

Since later CSS styles override any styles that come before, there is a preferred order to pseudo-class selectors: (Notice the different ways to specify color)


.class a:link {
color: rgb(248, 99, 99);
}

.class a:visited {
color: #A020F0;
}

.class a:focus {
color: hsl(174, 100%, 50%);
}

.class a:hover {
color: deeppink;
}

.class a:active {
color: var(--active-color);
}
